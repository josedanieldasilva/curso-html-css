para ficar do assim: 

            
    num=int(input('digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'número {num} é ÍPAR')
    print('fim do programa')
Utiliza a tag <pre>. Que deixa com mesma indentação ou com o mesmo espaço que está escrito no código.
