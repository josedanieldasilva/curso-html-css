Lista de Definições

Uma lista de definições fuciona como um dicionário, tedo um termo e sua discrição.

Formato

        
            <dl>(Lista de Difinição)
                <dt>(Termo)</dt>
                <dd>(Descrição)</dd>
            </dl>

EXEMPLO

HTML
Linguagem de marcação para criação de contéudo de site
CSS
Linguagem de marcação para design de um site.
javaScript
Linguagem de progamacão para a criacão de um site.
